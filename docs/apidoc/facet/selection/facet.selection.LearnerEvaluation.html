facet.selection.LearnerEvaluation¶

class facet.selection.LearnerEvaluation(pipeline, parameters, scoring_name, scores, ranking_score)[source]¶

A collection of scores for a specific parametrization of a learner pipeline, generated by a LearnerRanker.

Generic types

~T_LearnerPipelineDF(RegressorPipelineDF, ClassifierPipelineDF)

Parameters

  • pipeline (~T_LearnerPipelineDF) – the unfitted learner pipeline

  • parameters (Mapping[str, Any]) – the hyper-parameters for which the learner pipeline was scored, as a mapping of parameter names to parameter values

  • scores (ndarray) – the scores of all crossfits of the learner pipeline

  • ranking_score (float) – the aggregate score determined by the ranking metric of the LearnerRanker, used for ranking the learners
