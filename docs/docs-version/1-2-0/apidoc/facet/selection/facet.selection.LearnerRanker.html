facet.selection.LearnerRanker¶

class facet.selection.LearnerRanker(grids, cv, scoring=None, ranking_scorer=None, random_state=None, n_jobs=None, shared_memory=None, pre_dispatch=None, verbose=None)[source]¶

Score and rank different parametrizations of one or more learners, using cross-validation.

The learner ranker can run a simultaneous grid search across multiple alternative learner pipelines, supporting the ability to simultaneously select a learner algorithm and optimize hyper-parameters.

Parameters

  • grids (Union[LearnerGrid[~T_LearnerPipelineDF], Iterable[LearnerGrid[~T_LearnerPipelineDF]]]) – learner grids to be ranked (either a single grid, or an iterable of multiple grids)

  • cv (Optional[BaseCrossValidator]) – a cross validator (e.g., BootstrapCV)

  • scoring (Union[str, Callable[[float, float], float], None]) – a scoring function (by name, or as a callable) for evaluating learners (optional; use learner’s default scorer if not specified here). If passing a callable, the "score" will be used as the name of the scoring function unless the callable defines a __name__ attribute

  • ranking_scorer (Optional[Callable[[ndarray], float]]) – a function to calculate a scalar score for every crossfit and returning a float. The resulting score is used to rank all crossfits (highest score is best). Defaults to default_ranking_scorer(), calculating mean(scores) - 2 * std(scores, ddof=1)

  • random_state (Union[int, RandomState, None]) – optional random seed or random state for shuffling the feature column order

  • n_jobs (Optional[int]) – number of jobs to use in parallel; if None, use joblib default (default: None)

  • shared_memory (Optional[bool]) – if True, use threads in the parallel runs; if False, use multiprocessing (default: False)

  • pre_dispatch (Union[str, int, None]) – number of batches to pre-dispatch; if None, use joblib default (default: None)

  • verbose (Optional[int]) – verbosity level used in the parallel computation; if None, use joblib default (default: None)

static default_ranking_scorer(scores)[source]¶

The default function used to rank pipelines.

Calculates mean(scores) - 2 * std(scores, ddof=1), i.e., ranks pipelines by a (pessimistic) lower bound of the expected score.
