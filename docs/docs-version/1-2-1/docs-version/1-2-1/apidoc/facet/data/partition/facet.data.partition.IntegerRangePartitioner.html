facet.data.partition.IntegerRangePartitioner¶

class facet.data.partition.IntegerRangePartitioner(max_partitions=None, lower_bound=None, upper_bound=None)[source]¶

Partition integer values in adjacent intervals of the same length.

The range of intervals and interval size is computed based on attributes max_partitions, lower_bound, and upper_bound.

Partition boundaries and interval sized are chosen with interpretability in mind and are always an integer and a power of 10, or a multiple of 2 or 5 of a power of 10, e.g. 1, 2, 5, 10, 20, 50, and so on.

The intervals also satisfy the following conditions:

  • lower_bound is within the first interval

  • upper_bound is within the last interval

For example, with max_partitions = 5, lower_bound = 3, and upper_bound = 11, the resulting partitioning would be: [2, 4), [4, 6), [6, 8), [8, 10), [10, 12)
