facet.simulation.viz.SimulationMatplotStyle¶

class facet.simulation.viz.SimulationMatplotStyle(*, ax=None, colors=None)[source]¶

matplotlib style for simulation chart.

Along the range of simulated feature values on the x axis, plots the median and confidence intervals of the simulated target value.

A bar chart below the plot shows a histogram of actually observed values near the simulated values.
