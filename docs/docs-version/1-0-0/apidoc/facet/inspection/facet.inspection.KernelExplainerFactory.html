class facet.inspection.KernelExplainerFactory(link=None, l1_reg='num_features(10)', data_size_limit=100)[source]¶

A factory constructing class:~shap.KernelExplainer objects.

Bases: ExplainerFactory

Bases: ExplainerFactory

Parameters

  • link – (optional) override the default link parameter

  • l1_reg (Optional[str]) – (optional) override the default l1_reg parameter of method shap_values(); pass None to use the default value used by shap_values()

  • data_size_limit (Optional[int]) – (optional) maximum number of observations to use as the background data set; larger data sets will be down-sampled using method kmeans(). Pass None to prevent down-sampling the background data set.
